在本实验中，您将使用高级设计功能增强销售分析。

实验室故事

该实验室是一系列实验室中的众多实验室之一，这些实验室被设计成一个完整的故事，从数据准备到发布为报告和仪表板。您可以按任何顺序完成实验。但是，如果您打算通过多个实验室工作，我们建议您按以下顺序进行：

  7. 在 Power BI Desktop 中设计报表，第 2 部分

  9. 在 Power BI Desktop 中执行数据分析

  10. 实施行级安全

练习 1：配置同步切片器

在本练习中，您将同步报表页面切片器。

任务 1：开始 - 登录

在此任务中，你将通过登录 Power BI 设置实验室环境。

重要提示：如果你已登录 Power BI，请从下一个任务继续。

  1. 要打开 Microsoft Edge，请在任务栏上单击 Microsoft Edge 程序快捷方式。
  1. 在 Microsoft Edge 浏览器窗口中，导航到https://powerbi.microsoft.com。
提示：您还可以使用 Microsoft Edge 收藏夹栏中的 Power BI 服务收藏夹。
  1. 单击登录（位于右上角）。

  1. 输入提供给您的帐户详细信息。

  2. 如果提示更新密码，请重新输入提供的密码，然后输入并确认新密码。

重要提示：请务必记录您的新密码。

  1. 完成登录过程。

  2. 如果 Microsoft Edge 提示保持登录状态，请单击是。

  3. 在 Microsoft Edge 浏览器窗口的 Power BI 服务的导航窗格中，展开我的工作区。

  1. 让 Microsoft Edge 浏览器窗口保持打开状态。

任务 2：开始 - 打开报告

在此任务中，您将通过打开入门报告设置实验室环境。

重要提示：如果您从上一个实验继续（并且您成功完成了该实验），请不要完成此任务；相反，从下一个任务继续。

  1. 若要打开 Power BI Desktop，请在任务栏上单击 Microsoft Power BI Desktop 快捷方式。

  2. 要关闭入门窗口，请单击窗口左上角的X。

  3. 要登录 Power BI 服务，请单击右上角的登录。

  4. 使用用于登录 Power BI 服务的同一帐户完成登录过程。

  5. 若要打开初始 Power BI Desktop 文件，请单击“文件”功能区选项卡以打开后台视图。

  6. 选择打开报告。

  7. 单击浏览报告。

  8. 在打开窗口中，导航到D:\PL300\Labs\07-design-report-in-power-bi-desktop-enhanced\Starter文件夹。

  9. 选择销售分析文件。

  10. 单击打开。

  11. 关闭所有可能打开的信息窗口。

  12. 要创建文件的副本，请单击“文件”功能区选项卡以打开后台视图。

  13. 选择另存为。

  14. 如果提示应用更改，请单击应用。

  15. 在另存为窗口中，导航到D:\PL300\MySolution文件夹。

  16. 单击保存。

任务 3：同步切片器

在此任务中，您将同步年份和区域切片器。

你将继续开发在 Power BI Desktop 中设计报表，第 1 部分实验室中创建的报表。

  1. 在 Power BI Desktop 的“概述”页面上，将年份切片器设置为FY2018。

  2. 转到My Performance页面，然后注意Year切片器的值不同。

    当切片器未同步时，它可能会导致数据的误传和报告用户的挫败感。您现在将同步报表切片器。

  3. 返回概览页面，然后选择年份切片器。

  4. 在视图功能区选项卡上，从显示窗格组中，单击同步切片器。

  5. 在Sync Slicers窗格（在Visualizations窗格的左侧）的第二列（表示同步）中，选中Overview和My Performance页面的复选框。

  6. 在概览页面上，选择区域切片器。

  7. 将切片器与概览和利润页面同步。

  8. 通过选择不同的过滤器选项来测试同步切片器，然后验证同步切片器是否通过相同的选择进行过滤。

  9. 要关闭Sync Slicer页面，请单击位于窗格右上角的X。

练习 2：配置钻取

在本练习中，您将创建一个新页面并将其配置为钻取页面。完成设计后，页面将如下所示：

任务 1：创建钻取页面

在此任务中，您将创建一个新页面并将其配置为钻取页面。

  1. 添加一个名为Product Details的新报告页面。

  2. 右键单击产品详细信息页面选项卡，然后选择隐藏页面。

    报表用户将无法直接进入钻取页面。他们需要从其他页面上的视觉效果中访问它。在本实验的最后一个练习中，您将学习如何钻取到该页面。

  3. 在Visualizations窗格下，在Drill Through部分中，添加Product | 将类别字段添加到在此处添加钻取字段框。

    实验室使用速记符号来引用字段。它看起来像这样：产品 | 类别。在此示例中，Product是表名，Category是字段名。

  4. 要测试钻取页面，请在钻取过滤卡中选择Bikes。

  5. 请注意报告页面左上角的箭头按钮。

    将字段添加到钻孔/区域时会自动添加一个按钮。它允许报表用户导航回他们从中钻取的页面。

  6. 向页面添加卡片视觉对象，然后调整其大小和位置，使其位于按钮的右侧并填充页面的剩余宽度。

  7. 拖动产品 | 类别字段进入卡片视觉。

  8. 配置视觉对象的格式选项，然后将Category Label属性设置为Off。

  9. 将背景颜色属性设置为浅灰色。

  10. 将表格视觉对象添加到页面，然后调整其大小和位置，使其位于卡片视觉对象下方并填充页面上的剩余空间。

  11. 将以下字段添加到视觉对象：

    • 产品 | 子类别

    • 产品 | 颜色

    • 销售 | 数量

    • 销售 | 销售量

    • 销售 | 利润率

  12. 配置视觉对象的格式选项，并在Values部分中，将Text Size属性设置为20pt。

    钻取页面的设计几乎完成了。在下一个练习中，您将使用条件格式来增强页面。

练习 3：添加条件格式

在本练习中，您将使用条件格式增强钻取页面。完成设计后，页面将如下所示：

任务 1：添加条件格式

在此任务中，您将使用条件格式增强钻取页面。

  1. 选择表格视觉对象。

  2. 在可视化窗格中，单击利润率值上的向下箭头，然后选择条件格式 | 图标。

  3. 在Icons – Profit Margin窗口中，在Icon Layout下拉列表中，选择Right of Data。

  4. 要删除中间规则，请单击黄色三角形左侧的X。

  5. 配置第一条规则（红色菱形）如下：

    • 在第二个控件中，删除值

    • 在第三个控件中，选择Number

    • 在第五个控件中，输入0

    • 在第六个控件中，选择Number

  6. 配置第二条规则（绿色圆圈）如下：

    • 在第二个控件中，输入0

    • 在第三个控件中，选择Number

    • 在第五个控件中，去掉值

    • 在第六个控件中，选择Number

    规则可以解释为：如果利润率值小于0，则显示红色菱形；否则，如果该值大于或等于 0，则显示一个绿色圆圈。

  7. 单击确定。

  8. 在表格视觉对象中，验证是否显示了正确的图标。

  9. 为颜色字段配置背景颜色条件格式。

  10. 在Background Color – Color窗口的F​​ormat Style下拉列表中，选择Field Value。

  11. 我们应该基于什么领域？下拉列表，选择产品 | 格式化 | 背景颜色格式。

  12. 单击确定。

  13. 重复前面的步骤，使用Product |为Color字段配置字体颜色条件格式。格式化 | 字体颜色格式字段

    您可能还记得背景和字体颜色来自Power BI Desktop 中的准备数据实验室中的ColorFormats.csv文件，然后与Power BI Desktop 中的加载数据实验室中的产品查询集成。

练习 4：添加书签和按钮

在本练习中，您将使用按钮增强“我的绩效”页面，允许报表用户选择要显示的视觉类型。完成设计后，页面将如下所示：

任务 1：添加书签

在此任务中，您将添加两个书签，一个用于显示每个月销售额/目标视觉效果。

  1. 进入我的表现页面。

  2. 在视图功能区选项卡上，从显示窗格组内，单击书签。

  3. 在视图功能区选项卡上，从显示窗格组内单击选择。

  4. 在选择窗格中，在Sales 和 Target by Month项旁边，要隐藏视觉对象，请单击眼睛图标。

  5. 在书签窗格中，单击添加。

  6. 要重命名书签，请双击书签。

  7. 如果可见图表是条形图，则将书签重命名为Bar Chart ON，否则将书签重命名为Column Chart ON。

  8. 要编辑书签，在书签窗格中，将光标悬停在书签上，单击省略号，然后选择数据。

    禁用数据选项意味着书签不会使用当前的过滤器状态。这很重要，因为否则书签将永久锁定在Year切片器当前应用的过滤器中。

  9. 要更新书签，请再次单击省略号，然后选择更新。

    在以下步骤中，您将创建并配置第二个书签以显示第二个视觉对象。

  10. 在“选择”窗格中，切换两个Sales 和 Target by Month项目的可见性。

    换句话说，将可见的视觉隐藏起来，并使隐藏的视觉可见。

  11. 创建第二个书签，并适当命名（打开柱形图或打开条形图）。

  12. 将第二个书签配置为忽略过滤器（关闭数据选项），然后更新书签。

  13. 在“选择”窗格中，要使两个视觉对象都可见，只需显示隐藏的视觉对象。

  14. 调整两个视觉对象的大小和位置，使它们填满多卡视觉对象下方的页面，并相互完全重叠。

    提示：要选择被覆盖的视觉对象，请在“选择”窗格中选择它。

  15. 在“书签”窗格中，选择每个书签，并注意只有一个视觉对象可见。

    设计的下一阶段是向页面添加两个按钮，这将允许报表用户选择书签。

任务 2：添加按钮

在此任务中，您将添加两个按钮，并为每个按钮分配书签操作。

  1. 在“插入”功能区的“元素”组内，单击“按钮”，然后选择“空白” 。

  2. 将按钮直接放在年份切片器下方。

  3. 选择该按钮，然后在“格式”按钮窗格中，单击“常规”并将Title属性设置为On。

  4. 展开Title部分，然后在Text框中，输入Bar Chart。

  5. 展开背景部分，然后使用补色设置背景颜色。

  6. 单击Button并将Action属性设置为On。

  7. 展开Action部分，然后将Type下拉列表设置为Bookmark。

  8. 在书签下拉列表中，选择Bar Chart ON。

  9. 使用复制和粘贴创建按钮的副本，然后按如下方式配置新按钮：

    提示：复制和粘贴的快捷命令是Ctrl+C后跟Ctrl+V。

    • 将按钮文本属性设置为柱形图

    • 在“操作”部分中，将“书签”下拉列表设置为“打开柱形图”

    销售分析报告的设计现已完成。

任务 3：发布报告

在此任务中，您将发布报告。

  1. 选择概览页面。

  2. 在年份切片器中，选择FY2020。

  3. 在区域切片器中，选择全选。

  4. 保存 Power BI Desktop 文件。

    在发布到 Power BI 服务之前，必须始终保存该文件。

  5. 在“主页”功能区选项卡的“共享”组中，单击“发布” 。

  6. 在“发布到 Power BI”窗口中，请注意已选中“我的工作区”。

  7. 要发布报告，请单击选择。

  8. 如果提示替换数据集，请单击替换。

  9. 发布成功后，单击“知道了” 。

  10. 关闭 Power BI Desktop。

    你将在下一个练习中探索 Power BI 服务中的报表。

练习 5：探索报告

在本练习中，你将探索 Power BI 服务中的报表。

任务 1：浏览报告

在此任务中，你将探索 Power BI 服务中的报表。

  1. 在 Microsoft Edge 浏览器窗口的 Power BI 服务的导航窗格中，选择我的工作区，然后单击销售分析报告。

  2. 要测试追溯报告，请在“概览”页面的“按类别数量”视觉对象中，右键单击服装栏，然后选择追溯 | 产品详情。

  3. 请注意，产品详细信息页面适用于Clothing。

  4. 要返回源页面，请单击页面左上角的箭头按钮。

  5. 选择我的表现页面。

  6. 单击每个按钮，然后注意显示的是不同的视觉对象。

任务2：完成

在此任务中，您将完成实验。

  1. 要返回您的工作区，请在窗口网页的横幅中单击我的工作区。

  2. 让 Microsoft Edge 浏览器窗口保持打开状态。
