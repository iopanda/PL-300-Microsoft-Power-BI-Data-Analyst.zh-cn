创建 Power BI 仪表板

完成实验的预计时间为 45 分钟

在本实验中，您将创建销售监控仪表板。

在本实验中，您将学习如何：

  • 将视觉对象固定到仪表板

  • 使用问答创建仪表板磁贴

实验室故事

该实验室是一系列实验室中的众多实验室之一，这些实验室被设计成一个完整的故事，从数据准备到发布为报告和仪表板。您可以按任何顺序完成实验。但是，如果您打算通过多个实验室工作，我们建议您按以下顺序进行：

  1. 在 Power BI Desktop 中准备数据

  2. 在 Power BI Desktop 中加载数据

  3. Power BI Desktop 中的模型数据

  4. 在 Power BI Desktop 中创建 DAX 计算，第 1 部分

  5. 在 Power BI Desktop 中创建 DAX 计算，第 2 部分

  6. 在 Power BI Desktop 中设计报表，第 1 部分

  7. 在 Power BI Desktop 中设计报表，第 2 部分

  8. 创建 Power BI 仪表板

  9. 在 Power BI Desktop 中执行数据分析

  10. 实施行级安全

练习 1：创建仪表板

在本练习中，您将创建销售监控仪表板。完成的仪表板将如下所示：

任务 1：开始 - 登录

在此任务中，你将通过登录 Power BI 设置实验室环境。

重要提示：如果你已在之前的实验室中登录 Power BI，请继续下一个任务。

  1. 要打开 Microsoft Edge，请在任务栏上单击 Microsoft Edge 程序快捷方式。

  2. 在 Microsoft Edge 浏览器窗口中，导航到https://powerbi.microsoft.com。

    提示：您还可以使用 Microsoft Edge 收藏夹栏中的 Power BI 服务收藏夹。

  3. 单击登录（位于右上角）。

  4. 输入提供给您的帐户详细信息。

  5. 如果提示更新密码，请重新输入提供的密码，然后输入并确认新密码。

    重要提示：请务必记录您的新密码。

  6. 完成登录过程。

  7. 如果 Microsoft Edge 提示保持登录状态，请单击是。

  8. 在 Microsoft Edge 浏览器窗口的 Power BI 服务的导航窗格中，展开我的工作区。

  9. 让 Microsoft Edge 浏览器窗口保持打开状态。

任务 2：开始 - 打开报告

在此任务中，您将通过打开入门报告设置实验室环境。

重要提示：如果您从上一个实验继续（并且您成功完成了该实验），请不要完成此任务；相反，从下一个任务继续。

  1. 若要打开 Power BI Desktop，请在任务栏上单击 Microsoft Power BI Desktop 快捷方式。

  2. 要关闭入门窗口，请单击窗口左上角的X。

  3. 如果 Power BI Desktop 未登录 Power BI 服务，请单击右上角的登录。

  4. 使用用于登录 Power BI 服务的同一帐户完成登录过程。

  5. 若要打开初始 Power BI Desktop 文件，请单击“文件”功能区选项卡以打开后台视图。

  6. 选择打开报告。

  7. 单击浏览报告。

  8. 在打开窗口中，导航到D:\PL300\Labs\08-create-power-bi-dashboard\Starter文件夹。

  9. 选择销售分析文件。

  10. 单击打开。

  11. 关闭所有可能打开的信息窗口。

  12. 要创建文件的副本，请单击“文件”功能区选项卡以打开后台视图。

  13. 选择另存为。

  14. 如果提示应用更改，请单击应用。

  15. 在另存为窗口中，导航到D:\PL300\MySolution文件夹。

  16. 单击保存。

任务 3：开始 - 发布报告

在此任务中，您将通过创建数据集来设置实验室环境。

重要提示：如果你已在 Power BI Desktop 中设计报表，第 2 部分实验室中发布了报表，请继续下一个任务。

  1. 在 Microsoft Edge 浏览器窗口的 Power BI 服务中，在导航窗格的底部，单击获取数据。

  2. 在文件磁贴中，单击获取。

  3. 单击本地文件磁贴。

  4. 在打开窗口中，导航到D:\PL300\Labs\08-create-power-bi-dashboard\Solution文件夹。

  5. 选择Sales Analysis.pbix文件，然后单击Open。

  6. 如果提示替换数据集，请单击替换它。

任务 4：创建仪表板

在此任务中，您将创建销售监控仪表板。您将固定报表中的视觉对象，并基于图像数据 URI 添加磁贴，并使用问答创建磁贴。

  1. 在 Microsoft Edge 浏览器窗口的 Power BI 服务中，打开“销售分析”报表。

  2. 在概览页面中，将年份切片器设置为FY2020。

  3. 将区域切片器设置为Select All。

    将视觉对象固定到仪表板时，它们将使用当前过滤器上下文。固定后，无法更改过滤器上下文。对于基于时间的过滤器，最好使用相对日期切片器（或者，使用基于相对时间的问题进行问答）。

  4. 要创建仪表板并固定视觉对象，请将光标悬停在按月（列/行）视觉对象的销售额和利润率上。

  5. 在右下角，单击图钉。

  6. 在Pin to Dashboard窗口的Dashboard Name框中，输入Sales Monitoring。

  7. 单击固定。

  8. 打开导航窗格，选择我的工作区，然后打开销售监控仪表板。

  9. 请注意，仪表板只有一个磁贴。

  10. 要添加基于问题的磁贴，请在仪表板的左上角单击就您的数据提出问题。

    您可以使用问答功能提出问题，Power BI 将响应视觉对象。

  11. 单击蓝色框中的问答框下方的任何一个建议问题。

  12. 查看响应。

  13. 从问答框中删除所有文本。

  14. 在 Q&A 框中，输入以下内容：Sales YTD

  15. 注意(Blank)的响应。

    您可能还记得您在 Power BI Desktop 中创建 DAX 计算，第 2 部分实验室中添加了年初至今的销售额度量。此度量是时间智能表达式，因此需要对Date表进行筛选才能生成结果。

  16. 将问题扩展为：in year FY2020。

  17. 请注意，现在的响应是$33M。

  18. 要将响应固定到仪表板，请单击右上角的固定视觉对象。

  19. 当提示将磁贴固定到仪表板时，单击固定。

  20. 要返回仪表板，请单击左上角的Exit Q&A。

  21. 要添加公司徽标，请在菜单栏上单击Edit，然后选择Add a Tile。

    使用这种技术添加仪表板磁贴，您可以使用媒体来美化仪表板，包括 Web 内容、图像、格式丰富的文本框和视频（使用 YouTube 或 Vimeo 链接）。

  22. 在Add a Tile窗格（位于右侧）中，选择Image tile。

  23. 单击下一步。

  24. 在Add Image Tile窗格的URL框中，输入D:\PL300\Resources\AdventureWorksLogo_DataURL.txt文件中的完整 URL。

    您可以使用其 URL 嵌入图像，也可以使用数据 URL，该 URL 内嵌内容。

  25. 在窗格底部，单击应用。

  26. 要调整徽标图块的大小，请拖动右下角，然后将图块调整为宽一个单位，高两个单位。

    平铺尺寸被限制为矩形。只能将大小调整为矩形的倍数。

  27. 组织这些磁贴，使徽标出现在左上角，Sales YTD 磁贴在其下方，Sales, Profit Margin 磁贴在右侧。

任务 5：编辑磁贴详细信息

在此任务中，您将编辑两个图块的详细信息。

  1. 将光标悬停在Sales YTD 磁贴上，然后在磁贴的右上角单击省略号，然后选择Edit Details。

  2. 在Tile Details窗格（位于右侧）的Subtitle框中，输入FY2020。

  3. 单击应用。

  4. 请注意，Sales YTD 磁贴显示了一个副标题。

  5. 编辑销售、利润率磁贴的磁贴详细信息。

  6. 在磁贴详细信息窗格的功能部分中，选中显示上次刷新时间。

  7. 单击应用。

  8. 请注意，磁贴描述了上次刷新时间（在 Power BI Desktop 中加载数据模型时完成）。

    您将在下一个练习中刷新数据集。通常，这将通过使用计划刷新来实现，在这种情况下，Power BI 将使用网关连接到 SQL Server 数据库。但是，由于教室设置的限制，没有网关。因此，你将打开 Power BI Desktop，执行手动数据刷新，然后将文件上传到你的工作区。

练习 2：刷新数据集

在本练习中，您将首先将 2020 年 6 月的销售订单数据加载到AdventureWorksDW2020数据库中。然后，您将打开您的 Power BI Desktop 文件，执行数据刷新，然后将文件上传到您的工作区。

任务 1：更新实验室数据库

在此任务中，您将运行 PowerShell 脚本来更新AdventureWorksDW2020数据库中的数据。

  1. 在文件资源管理器的D:\PL300\Setup文件夹中，右键单击UpdateDatabase-2-AddSales.ps1文件，然后选择使用 PowerShell 运行。

  2. 如果提示更改执行策略，请按A。

  3. 当提示按任意键关闭时，再次按Enter。

    AdventureWorksDW2020数据库现在包括 2020 年 6 月的销售订单。

任务 2：刷新 Power BI Desktop 文件

在此任务中，您将打开销售分析Power BI Desktop 文件，执行数据刷新，然后将文件上传到您的销售分析工作区。

  1. 在 Power BI Desktop 文件的“字段”窗格中，右键单击“销售”表，然后选择“刷新数据” 。

  2. 刷新完成后，保存 Power BI Desktop 文件。

  3. 要将文件发布到您的工作区，请在“主页”功能区选项卡上的“共享”组中，单击“发布” ，然后单击“选择发布”。

  4. 当提示替换数据集时，单击替换。

    Power BI 服务中的数据集现在包含 2020 年 6 月的销售数据。

  5. 关闭 Power BI Desktop。

练习 3：查看仪表板

在本练习中，您将查看仪表板以了解更新的销售情况。

任务 1：查看仪表板

在此任务中，您将查看仪表板以注意更新的销售。

  1. 在 Microsoft Edge 浏览器窗口的 Power BI 服务中，查看销售监控仪表板。

  2. 在Sales, Profit Margin 磁贴的副标题中，请注意数据已刷新NOW。

  3. 另请注意，现在有一列2020 Jun。

    如果您没有看到 2020 年 6 月的数据，您可能需要按F5重新加载 Web 浏览器。

  4. 要关闭窗格，请单击关闭。
