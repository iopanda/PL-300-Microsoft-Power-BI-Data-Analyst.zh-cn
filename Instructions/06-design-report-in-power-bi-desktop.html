在 Power BI Desktop 中设计报表，第 1 部分

完成实验的预计时间为 45 分钟

在本实验中，您将创建一个三页的报告。然后，您将其发布到 Power BI，然后您将打开该报表并与之交互。

在本实验中，您将学习如何：

  • 设计报告

  • 配置视野和格式属性

实验室故事

该实验室是一系列实验室中的众多实验室之一，这些实验室被设计成一个完整的故事，从数据准备到发布为报告和仪表板。您可以按任何顺序完成实验。但是，如果您打算通过多个实验室工作，我们建议您按以下顺序进行：

  1. 在 Power BI Desktop 中准备数据

  2. 在 Power BI Desktop 中加载数据

  3. Power BI Desktop 中的模型数据

  4. 在 Power BI Desktop 中创建 DAX 计算，第 1 部分

  5. 在 Power BI Desktop 中创建 DAX 计算，第 2 部分

  6. 在 Power BI Desktop 中设计报表，第 1 部分

  7. 在 Power BI Desktop 中设计报表，第 2 部分

  8. 创建 Power BI 仪表板

  9. 在 Power BI Desktop 中执行数据分析

  10. 实施行级安全

练习 1：创建报告

在本练习中，您将创建一个名为Sales Report的三页报告。

任务 1：开始 - 登录

在此任务中，你将通过登录 Power BI 设置实验室环境。

重要提示：如果你已登录 Power BI，请从下一个任务继续。

  1. 要打开 Microsoft Edge，请在任务栏上单击 Microsoft Edge 程序快捷方式。
  1. 在 Microsoft Edge 浏览器窗口中，导航到https://powerbi.microsoft.com。
提示：您还可以使用 Microsoft Edge 收藏夹栏中的 Power BI 服务收藏夹。
  1. 单击登录（位于右上角）。

  1. 输入提供给您的帐户详细信息（查看资源）。

  2. 如果提示更新密码，请重新输入提供的密码，然后输入并确认新密码。

重要提示：请务必记录您的新密码。

  1. 完成登录过程。

  2. 如果 Microsoft Edge 提示保持登录状态，请单击是。

  3. 让 Microsoft Edge 浏览器窗口保持打开状态。

任务 2：开始 - 启用地图和填充的地图视觉效果

在此任务中，您将通过更新 Power BI 管理门户中的集成设置在实验室环境中启用地图和填充地图视觉对象。

  1. 若要打开 Power BI 管理员门户，请在浏览器的右上角单击“设置”图标。
  1. 选择管理员门户。
  1. 向下滚动页面到集成设置。单击箭头以展开地图和填充地图视觉效果选项。

  1. 将地图和填充地图视觉效果选项设置为Enabled。

  2. 单击Apply应用更改。

  3. 浏览器右上角将显示一条消息，说明租户设置更改将在接下来的 15 分钟内应用。

  4. 让 Microsoft Edge 浏览器窗口保持打开状态。

任务 3：开始 - 打开报告

在此任务中，您将通过打开入门报告设置实验室环境。

重要提示：如果您从上一个实验继续（并且您成功完成了该实验），请不要完成此任务；相反，从下一个任务继续。

  1. 若要打开 Power BI Desktop，请在任务栏上单击 Microsoft Power BI Desktop 快捷方式。

  2. 要关闭入门窗口，请单击窗口左上角的X。

  3. 要登录 Power BI 服务，请单击右上角的登录。

  4. 使用用于登录 Power BI 服务的同一帐户完成登录过程。

  5. 若要打开初始 Power BI Desktop 文件，请单击“文件”功能区选项卡以打开后台视图。

  6. 选择打开报告。

  7. 单击浏览报告。

  8. 在打开窗口中，导航到D:\PL300\Labs\06-design-report-in-power-bi-desktop\Starter文件夹。

  9. 选择销售分析文件。

  10. 单击打开。

  11. 关闭所有可能打开的信息窗口。

  12. 要创建文件的副本，请单击“文件”功能区选项卡以打开后台视图。

  13. 选择另存为。

  14. 如果提示应用更改，请单击应用。

  15. 在另存为窗口中，导航到D:\PL300\MySolution文件夹。

  16. 单击保存。

任务 4：设计页面 1

在此任务中，您将设计第一个报告页面。完成设计后，页面将如下所示：

  1. 在 Power BI Desktop 中，要重命名页面，请在左下角右键单击Page 1，然后选择Rename。

    提示：您也可以双击页面名称进行重命名。

  2. 将页面重命名为Overview，然后按Enter。

  3. 要添加图像，请在插入功能区选项卡上的元素组内单击图像。

  4. 在打开窗口中，导航到D:\PL300\Resources文件夹。

  5. 选择AdventureWorksLogo.jpg文件，然后单击打开。

  6. 拖动图像以将其放置在左上角，并拖动引导标记以调整其大小。

  7. 要添加切片器，首先通过单击报告页面的空白区域取消选择图像。

  8. 在字段窗格中，选择日期 | 年份字段（不是层次结构的年份级别）。

    实验室使用速记符号来引用字段。它看起来像这样：日期 | 年。在此示例中，Date是表名，Year是字段名。

  9. 请注意，已将年份值表添加到报告页面。

  10. 要将视觉对象从表转换为切片器，请在Visualizations窗格中选择Slicer。

  11. 要将切片器从列表转换为下拉列表，请单击切片器右上角的向下箭头，然后选择下拉菜单。

  12. 调整切片器的大小和位置，使其位于图像下方并且与图像的宽度相同。

  13. 在年份切片器中，打开下拉列表，选择FY2020，然后折叠下拉列表。

    报告页面现在按FY2020年过滤。

  14. 通过单击报告页面的空白区域取消选择切片器。

  15. 根据Region |创建第二个切片器 Region字段（不是层次结构的Region级别）。

  16. 将切片器保留为列表，然后调整切片器的大小并将其放置在Year切片器下方。

  17. 通过单击报告页面的空白区域取消选择切片器。

  18. 要将图表添加到页面，请在“可视化”窗格中，单击“折线图和堆积柱形图”视觉类型。

  19. 调整视觉对象的大小和位置，使其位于徽标的右侧，从而填充报告页面的宽度。

  20. 将以下字段拖放到视觉对象中：

    • 日期 | 月

    • 销售 | 销售量

  21. 在可视字段窗格（不是字段窗格——可视字段窗格位于可视化窗格下方）中，请注意字段已分配给X 轴和列 y 轴孔/区域。

    通过将字段拖到视觉对象中，它们将被添加到默认井/区域。为了精确起见，您可以将字段直接拖到井/区域中，就像您接下来要做的那样。

  22. 从字段窗格中，拖动销售 | 利润率字段进入Y 轴线井/区域。

  23. 请注意，视觉对象只有 11 个月。

    今年的最后一个月，即 2020 年 6 月，还没有任何销售。默认情况下，视觉已经消除了几个月的空白销售。您现在将配置视觉对象以显示所有月份。

  24. 在可视字段窗格的X 轴井/区域中，对于Month字段，单击向下箭头，然后选择Show Items With No Data。

  25. 请注意，现在出现了2020 年 6 月。

  26. 通过单击报告页面的空白区域取消选择图表。

  27. 要将图表添加到页面，请在可视化窗格中，单击地图视觉对象类型。

  28. 调整视觉对象的大小和位置，使其位于柱形图/折线图下方，并填充上方图表宽度的一半。

  29. 将以下字段添加到可视井/区域：

    • 地点：地区 | 国家

    • 图例：产品 | 类别

    • 尺寸:销售 | 销售量

  30. 通过单击报告页面的空白区域取消选择图表。

  31. 要将图表添加到页面，请在“可视化”窗格中，单击“堆叠条形图”视觉类型。

  32. 调整视觉对象的大小和位置，使其填充剩余的报表页面空间。

  33. 将以下字段添加到可视井/区域：

    • 轴：产品 | 类别

    • 价值：销售额 | 数量

  34. 要格式化视觉对象，请打开“格式”窗格。

  35. 展开条形图，然后展开颜色组，然后将默认颜色属性设置为合适的颜色（以补充柱形图/折线图）。

  36. 将数据标签属性设置为On。

  37. 保存 Power BI Desktop 文件。

    第一页的设计现已完成。

任务 5：设计页面 2

在此任务中，您将设计第二个报告页面。完成设计后，页面将如下所示：

重要提示：当实验室中已经提供了详细说明时，实验室步骤将提供更简洁的说明。如果您需要详细说明，可以参考本实验中的其他任务。

  1. 要创建新页面，请单击左下角的加号图标。

  2. 将页面重命名为Profit。

  3. 添加基于Region 的切片器 | 区域字段。

  4. 使用“格式”窗格启用“全选”选项（在“选择”组中）。

  5. 调整切片器的大小和位置，使其位于报表页面的左侧，大约是页面高度的一半。

  6. 添加矩阵视觉对象，并调整其大小和位置，使其填充报告页面的剩余空间

  7. 添加日期 | 矩阵行井/面积的财政层次结构。

  8. 将以下五个销售表字段添加到值井/区域：

    • 订单（来自计数文件夹）

    • 销售量

    • 成本

    • 利润

    • 利润率

  9. 在“过滤器”窗格（位于“可视化”窗格的左侧）中，请注意“此页面上的过滤器”井/区域（您可能需要向下滚动）。

  10. 从字段窗格中，拖动产品 | 类别字段到此页面上的过滤器井/区域。

  11. 在过滤器卡片的右上角，单击箭头以折叠卡片。

    添加到“过滤器”窗格的字段可以获得与切片器相同的结果。一个区别是它们不占用报告页面上的空间。另一个区别是它们可以配置为实现更复杂的过滤要求。

  12. 将以下每个产品表字段添加到此页面上的过滤器井/区域，折叠每个，直接在类别卡下方：

    • 子类别

    • 产品

    • 颜色

  13. 保存 Power BI Desktop 文件。

    第二页的设计现已完成。

任务 6：设计页面 3

在此任务中，您将设计第三个（也是最后一个）报告页面。完成设计后，页面将如下所示：

  1. 创建一个新页面，然后将其重命名为My Performance。

  2. 要模拟行级安全过滤器的性能，请将Salesperson (Performance) | 销售人员字段到筛选器窗格中的页面级筛选器。

  3. 选择迈克尔·布莱斯。现在将过滤“我的绩效”报告页面上的数据，以仅显示 Michael Blythe 的数据。

  4. 添加基于日期的下拉切片器 | Year字段，然后调整其大小和位置，使其位于页面的左上角。

  5. 在切片器中，将页面设置为按FY2019过滤。

  6. 添加一个多行卡片视觉对象，然后调整其大小并重新定位它，使其位于切片器的右侧并填充页面的剩余宽度。

  7. 将以下四个字段添加到视觉对象：

    • 销售 | 销售量

    • 目标 | 目标

    • 目标 | 方差

    • 目标 | 方差保证金

  8. 格式化视觉：

    • 在标注值组中，将文本大小属性增加到28pt

    • 在背景组中，将颜色设置为浅灰色

  9. 添加群集条形图视觉对象，然后调整其大小和位置，使其位于多行卡片视觉对象下方并填充页面的剩余高度和多行卡片视觉对象宽度的一半。

  10. 将以下字段添加到可视井/区域：

    • 轴：日期 | 月

    • 价值：销售额 | 销售和目标 | 目标

  11. 要创建视觉对象的副本，请按Ctrl+C，然后按Ctrl+V。

  12. 将新视觉对象放置在原始视觉对象的右侧。

  13. 要修改可视化类型，请在Visualizations窗格中，选择Clustered Column Chart。

    现在可以看到由两种不同的可视化类型表示的相同数据。这不是页面布局的良好用途，但是，你将在 Power BI Desktop 中的设计报表，第 2 部分实验室中通过叠加视觉对象来改进它。通过向页面添加按钮，您将允许报表用户确定两个视觉对象中的哪一个是可见的。

    第三页也是最后一页的设计现已完成。

任务 7：发布报告

在此任务中，您将发布报告。

  1. 选择概览页面。

  2. 保存 Power BI Desktop 文件。

  3. 在“主页”功能区选项卡的“共享”组中，单击“发布” 。

  4. 在“发布到 Power BI”窗口中，请注意已选中“我的工作区”。

  5. 要发布报告，请单击选择。

  6. 发布成功后，单击“知道了” 。

  7. 使 Power BI Desktop 保持打开状态。

    你将在下一个练习中探索 Power BI 服务中的报表。

练习 2：探索报告

在本练习中，您将探索已发布到 Power BI 的报表。

任务 1：浏览报告

在此任务中，你将浏览已发布到 Power BI 的报表。

  1. 在 Microsoft Edge 浏览器窗口的 Power BI 服务的导航窗格（位于左侧，可以折叠）中，展开My Workspace。

  2. 查看工作区的内容，注意销售分析报告和数据集。

    发布 Power BI Desktop 文件时，数据模型作为数据集发布。

    如果您没有看到它，请按F5重新加载浏览器，然后再次展开工作区。

  3. 要打开报告，请单击销售分析报告。

  4. 在左侧的“页面”窗格中，选择“概览”页面。

  5. 在区域切片器中，按住Ctrl键的同时选择多个区域。

  6. 在柱/折线图中，选择任意月份列以交叉过滤页面。

  7. 在按住Ctrl键的同时，选择一个额外的月份。

    默认情况下，交叉筛选会筛选页面上的所有其他视觉对象。

  8. 请注意，条形图被过滤并突出显示，条形的粗体部分代表过滤的月份。

  9. 将光标悬停在条形图视觉对象上，然后在右上角将光标悬停在过滤器图标上。

    筛选器图标可让您了解应用于视觉对象的所有筛选器，包括切片器和来自其他视觉对象的交叉筛选器。

  10. 将光标悬停在条上，然后注意工具提示信息。

  11. 要撤消交叉过滤器，请在柱形图/折线图中单击视觉对象的空白区域。

  12. 将光标悬停在地图视觉对象上，然后在右上角单击焦点模式图标。

    焦点模式将视觉对象缩放到整页大小。

  13. 将光标悬停在条形图的不同部分以显示工具提示。

  14. 要返回报告页面，请点击左上角的Back to Report。

  15. 再次将光标悬停在地图视觉对象上，然后在右上角单击省略号 (...)，然后注意菜单选项。

  16. 尝试每个选项，但Teams 中的聊天除外。

  17. 在左侧的“页面”窗格中，选择“利润”页面。

  18. 请注意，区域切片器与概览页面上的区域切片器具有不同的选择。

    切片器不同步。你将修改报表设计以确保它们在 Power BI Desktop 中设计报表，第 2 部分实验室中的页面之间同步。

  19. 在“过滤器”窗格（位于右侧）中，展开过滤器卡并应用一些过滤器。

    “过滤器”窗格允许您定义比页面上可能容纳的更多的过滤器作为切片器。

  20. 在矩阵视觉对象中，使用加号 (+) 按钮深入了解财务层次结构。

  21. 选择我的表现页面。

  22. 在菜单栏的右上角，单击查看，然后选择全屏。

  23. 通过修改切片器与页面交互，并交叉过滤页面。

  24. 在窗口底部，注意更改页面、在页面之间向后或向前导航或退出全屏模式的命令。

  25. 单击左侧图标退出全屏模式。

任务2：完成

在此任务中，您将完成实验。

  1. 要返回您的工作区，请在窗口网页的横幅中单击我的工作区。

  2. 让 Microsoft Edge 浏览器窗口保持打开状态。

    您将在 Power BI Desktop 中设计报表，第 2 部分实验室中使用高级功能增强报表设计。
