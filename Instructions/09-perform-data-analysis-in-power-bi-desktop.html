在 Power BI Desktop 中执行数据分析

完成实验的预计时间为 45 分钟

在本实验中，您将创建销售探索报告。

在本实验中，您将学习如何：

  • 创建动画散点图

  • 使用视觉来预测值

实验室故事

该实验室是一系列实验室中的众多实验室之一，这些实验室被设计成一个完整的故事，从数据准备到发布为报告和仪表板。您可以按任何顺序完成实验。但是，如果您打算通过多个实验室工作，我们建议您按以下顺序进行：

  1. 在 Power BI Desktop 中准备数据

  2. 在 Power BI Desktop 中加载数据

  3. Power BI Desktop 中的模型数据

  4. 在 Power BI Desktop 中创建 DAX 计算，第 1 部分

  5. 在 Power BI Desktop 中创建 DAX 计算，第 2 部分

  6. 在 Power BI Desktop 中设计报表，第 1 部分

  7. 在 Power BI Desktop 中设计报表，第 2 部分

  8. 创建 Power BI 仪表板

  9. 在 Power BI Desktop 中执行数据分析

  10. 实施行级安全

练习 1：创建报告

在本练习中，您将创建销售探索报告。

任务 1：开始 - 登录

在此任务中，你将通过登录 Power BI 设置实验室环境。

重要提示：如果你已在之前的实验室中登录 Power BI，请继续下一个任务。

  1. 要打开 Microsoft Edge，请在任务栏上单击 Microsoft Edge 程序快捷方式。
  1. 在 Microsoft Edge 浏览器窗口中，导航到https://powerbi.microsoft.com。
提示：您还可以使用 Microsoft Edge 收藏夹栏中的 Power BI 服务收藏夹。
  1. 单击登录（位于右上角）。

  1. 输入提供给您的帐户详细信息。

  2. 如果提示更新密码，请重新输入提供的密码，然后输入并确认新密码。

重要提示：请务必记录您的新密码。

  1. 完成登录过程。

  2. 如果 Microsoft Edge 提示保持登录状态，请单击是。

  3. 在 Microsoft Edge 浏览器窗口的 Power BI 服务的导航窗格中，展开我的工作区。

  1. 让 Microsoft Edge 浏览器窗口保持打开状态。

任务 2：开始 - 创建数据集

在此任务中，您将通过创建数据集来设置实验室环境。

重要提示：如果你已经在创建 Power BI 仪表板实验室中发布了数据集，请继续下一个任务。

  1. 在 Microsoft Edge 浏览器窗口的 Power BI 服务中，在导航窗格的底部，单击获取数据。

  2. 在文件磁贴中，单击获取。

  3. 单击本地文件磁贴。

  4. 在打开窗口中，导航到D:\PL300\Labs\08-create-power-bi-dashboard\Solution文件夹。

  5. 选择Sales Analysis.pbix文件，然后单击Open。

  6. 如果提示替换数据集，请单击替换它。

任务 3：创建报告

在此任务中，您将创建销售探索报告。

  1. 若要打开 Power BI Desktop，请在任务栏上单击 Microsoft Power BI Desktop 快捷方式。

    重要提示：如果已打开 Power BI Desktop（来自以前的实验室），请关闭该实例。

  2. 要关闭入门窗口，请单击窗口左上角的X。

  3. 如果 Power BI Desktop 未登录 Power BI 服务，请单击右上角的登录。

  4. 使用用于登录 Power BI 服务的同一帐户完成登录过程。

  5. 要保存文件，请单击“文件”功能区选项卡以打开后台视图。

  6. 选择保存。

  7. 在另存为窗口中，导航到D:\PL300\MySolution文件夹。

  8. 在文件名框中，输入Sales Exploration并单击保存。

  9. 要创建与销售分析数据集的实时连接，请在“主页”功能区选项卡上的“数据”组内，单击“Power BI 数据集” 。

  10. 在Select a Dataset to Create a Report窗口中，选择Sales Analysis数据集。

  11. 单击创建。

  12. 保存 Power BI Desktop 文件。

    您现在将创建两个报表页面，并且在每个页面上您将使用不同的视觉对象来分析和探索数据。

练习 2：创建散点图

在本练习中，您将创建一个可以设置动画的散点图。

任务 1：创建动画散点图

在此任务中，您将创建一个可以设置动画的散点图。

  1. 将第 1 页重命名为Scatter Chart。

  2. 将散点图视觉对象添加到报表页面，然后对其进行定位和调整大小，使其填满整个页面。

  3. 将以下字段添加到可视井/区域：

    实验室使用速记符号来引用字段。它看起来像这样：经销商 | 业务类型。在此示例中，Reseller是表名称，Business Type是字段名称。

    • X 轴：销售 | 销售量

    • Y 轴：销售 | 利润率

    • 图例：经销商 | 业务类型

    • 尺寸:销售 | 数量

    • 播放轴：日期 | 四分之一

    将字段添加到Play Axis井/区域时，可以为图表设置动画。

  4. 在过滤器窗格中，添加产品 | 此页面上的过滤器井/区域的类别字段。

  5. 在过滤卡中，按Bikes过滤。

  6. 要为图表设置动画，请单击左下角的Play。

  7. 观看从FY2018 Q1到FY2020 Q4的整个动画周期。

    散点图允许同时理解度量值：在这种情况下，订单数量、销售收入和利润率。

    每个气泡代表一个经销商业务类型。气泡大小的变化反映了订单数量的增加或减少。水平移动代表销售收入的增加/减少，垂直移动代表盈利能力的增加/减少。

  8. 当动画停止时，单击其中一个气泡以显示其随时间的跟踪。

  9. 将光标悬停在任何气泡上以显示一个工具提示，该工具提示描述了该时间点经销商类型的度量值。

  10. 在Filters窗格中，仅按Clothing进行过滤，并注意它会产生非常不同的结果。

  11. 保存 Power BI Desktop 文件。

练习 3：创建预测

在本练习中，您将创建预测以确定未来可能的销售收入。

任务 1：创建预测

在此任务中，您将创建预测以确定未来可能的销售收入。

  1. 添加一个新页面，然后将该页面重命名为Forecast。

  2. 将折线图视觉对象添加到报表页面，然后定位并调整其大小以填充整个页面。

  3. 将以下字段添加到可视井/区域：

    • X 轴：日期 | 日期

    • Y 轴：销售额 | 销售量

  4. 在过滤器窗格中，添加日期 | 此页面上的过滤器井/区域的年份字段。

  5. 在过滤卡中，按两年过滤：FY2019和FY2020。

    在时间线上进行预测时，您将需要至少两个周期（年）的数据才能产生准确而稳定的预测。

  6. 还添加产品 | 类别字段到此页面上的过滤器井/区域，并按Bikes过滤。

  7. 要添加预测，请在“可视化”窗格下方选择“分析”窗格。

  8. 展开预测部分。

    如果“预测”部分不可用，可能是因为未正确配置视觉对象。预测仅在满足两个条件时可用：轴有一个日期类型的字段，并且只有一个值字段。

  9. 将“预测”选项设置为“开”。

  10. 配置以下预测属性：

    • 单位：月

    • 预测时长：1个月

    • 季节性：365

    • 置信区间：80%

  11. 单击应用。

  12. 在线条视觉中，请注意预测已超出历史数据一个月。

    灰色区域代表置信度。置信度越大，预测就越不稳定，因此也就越不准确。

    当您知道周期的长度时，在这种情况下为年度，您应该输入季节性点。有时可能是每周 (7) 或每月 (30)。

  13. 在Filters窗格中，仅按Clothing进行过滤，并注意它会产生不同的结果。

  14. 保存 Power BI Desktop 文件。

任务2：完成

在此任务中，您将完成实验。

  1. 选择散点图页面。

  2. 保存 Power BI Desktop 文件。

  3. 要将文件发布到“我的工作区”，请在“主页”功能区选项卡上的“共享”组中，单击“发布” ，然后单击“选择发布”。

  4. 关闭 Power BI Desktop。
